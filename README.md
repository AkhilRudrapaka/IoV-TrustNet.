# 🚨 C2S-VANET-IoT  
## Cooperative, Confirmation-Based, Severity-Aware Emergency Response System using VANET-IoT

---

## 📌 Abstract

Road accidents cause significant loss of life due to delayed emergency response and false accident reporting. This project proposes a **Cooperative, Confirmation-Based, Severity-Aware Emergency Response System** that integrates **Vehicular Ad Hoc Networks (VANET)** with **IoT and Edge Computing** to enable reliable, real-time accident detection and intelligent emergency handling.

The system utilizes vehicle sensor data (accelerometer, GPS) to detect crash events, verifies incidents through cooperative vehicle confirmation (V2V communication), classifies accident severity, and notifies emergency services through Road Side Units (RSUs) and cloud infrastructure. The proposed architecture minimizes false alarms, reduces response time, and enhances road safety in smart city environments.

---

## 🎯 Objectives

- Detect accidents using onboard vehicle sensors  
- Implement cooperative confirmation via V2V communication  
- Reduce false positives using multi-vehicle validation  
- Classify accident severity levels  
- Notify emergency services based on severity priority  
- Enable low-latency processing using edge computing  
- Integrate cloud-based monitoring and analytics  

---

## 🏗 System Architecture

The system is divided into four main layers:

### 1️⃣ Vehicle Layer
- On-Board Unit (OBU)  
- Accelerometer Sensor  
- GPS Module  
- Local Severity Estimation  

### 2️⃣ VANET Communication Layer
- V2V (Vehicle-to-Vehicle) Communication  
- V2I (Vehicle-to-Infrastructure) Communication  
- Road Side Units (RSUs)  

### 3️⃣ Edge / IoT Processing Layer
- IoT Gateway  
- Message Authentication  
- Multi-vehicle Confirmation Engine  
- Severity Classification Module  

### 4️⃣ Cloud Layer
- Accident Database  
- AI-based Analysis  
- Emergency Services Notification  
- Smart Traffic Signal Control  

---

## 🔄 Working Flow

1. Accident detected using accelerometer threshold  
2. GPS captures location coordinates  
3. Alert broadcasted to nearby vehicles (V2V)  
4. Nearby vehicles confirm the event  
5. RSU collects confirmed data via V2I  
6. Edge layer classifies severity:
   - Minor  
   - Moderate  
   - Severe  
7. Cloud server updates accident database  
8. Emergency services notified based on severity  
9. Traffic signals adjusted if required
